# Contributing to WordPress Inertia React Starter Theme

Thanks for your interest! Here's how to contribute:

1. Fork the repository.
2. Create a feature branch: `git checkout -b feature/your-feature`.
3. Commit changes: `git commit -m "Add your feature"`.
4. Push to your fork: `git push origin feature/your-feature`.
5. Open a Pull Request.

## Guidelines
- Keep code clean and documented.
- Test changes in a WordPress environment.
- Follow the existing structure (e.g., `assets/src/` for frontend).

## Issues
Feel free to report bugs or suggest enhancements via the Issues tab!
